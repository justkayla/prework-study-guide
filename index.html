HTML

  • The acronym HTML stands for Hypertext Markup Language.
  • HTML is one of the three cornerstone languages used in web development.
  • HTML is used to create the structure of a webpage.
  • HTML is made up of elements which work together to provide content and the basic framework for all pages on the web.
  • Text, Image, Header, and List elements are commonly used to add content to pages.
  • Meta-related, Content division, and Semantic elements are commonly used to divide the content on the page and to describe the content to browsers and other developers.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • The acronym CSS stands for Cascading Style Sheets.
  • CSS is a programming language used by developers to define how webpage content is presented to users.
  • CSS defines the colors, aesthetics, visual layout, and organization of a webpage.
  • CSS also defines style rules for specific elements of an HTML page and formats them using a wide range of style options.
  • There are three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet.
  • Inline CSS styles apply to a specific HTML tag, using a style attribute with a CSS rule to style a particular page element only.
  • Inline styles are useful for quick and permanent changes, but are less flexible than internal/external style sheets.
  • Each inline style must be edited individually whenever a design change is made.
  • An internal CSS style sheet contains CSS rules for the webpage in particular HTML elements.
  • An external CSS style sheet is the most common way to style a webpage.
  • An external CSS style sheet allows developers to keep all their CSS rules in a separate file, which makes design changes easier.
  • Two important CSS terms are: margin and padding.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • Git is the version control system that's responsible for everything GitHub related that happens locally on your computer.
  • There are a lot of git commands that are used in coding.
  • An example of a git command is git clone.
  • git clone: clones a remote repository into a new directory on your local drive.
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout branch-name: changes the branch
  • Use online git command cheat sheets to become familiar with git commands!

JavaScript

  • JavaScript (JS) is a high-level dynamic programming language that allows users to interact with websites.
  • With JS, you can power your websites with effects, interactivity, data communication, and much more!
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • JS reads code in order...so it's important to use the correct order.
  • You can view console.log output by right-clicking your index.html in the browser and selecting "Inspect" from the menu.
  • Emojis have character codes (you can look these up)!
  • Remember that you have to link the script.js file to the index.html file like you did with style.css!
